# Flujo: Comparación de Documentos Legales

## Objetivo
Identificar sistemáticamente diferencias, similitudes y aspectos críticos entre
dos o más documentos legales, presentando el análisis en formato estructurado.

## Restricciones
- NUNCA opinar sobre cuál documento es "mejor" sin criterio objetivo explícito
- NUNCA omitir diferencias en cláusulas de derechos, obligaciones o penalidades
- Siempre referenciar cada diferencia con (Documento A, Cláusula X) vs (Documento B, Cláusula Y)
- Si los documentos son de naturaleza distinta: advertirlo antes de comparar
- NO inventar cláusulas ausentes — registrar ausencia como tal

## Protocolo de Acción
1. Identificar el tipo de cada documento (contrato, estatuto, escritura, etc.)
2. Mapear las cláusulas homólogas entre documentos
3. Señalar diferencias en: plazos, montos, obligaciones, penalidades, condiciones resolutorias,
   jurisdicción, garantías, confidencialidad
4. Indicar cláusulas presentes en un documento pero ausentes en el otro
5. Presentar análisis en tabla comparativa y resumen narrativo de diferencias críticas

## Criterio de Éxito
Tabla comparativa completa con diferencias identificadas y referenciadas por documento y cláusula.
Resumen destacando las 3 diferencias más relevantes para el abogado.
